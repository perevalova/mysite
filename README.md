# My site

This is my personal website with information about me and my portfolio.

## Installation

#### 1. Install dependencies:

```bash
pip install -r requirements.txt
```

#### 2. Make migrations:

```bash
python manage.py migrate
```

#### 3. Run server:

```bash
python manage.py runserver
```
